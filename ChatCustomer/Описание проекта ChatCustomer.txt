ChatCustomer 2-я часть проекта
desktop Приложение WPF платформа .net Core 3.1
реализован патерн MVVM

Используемые библиотеки Newtonsoft.Json

Запуск проекта.
Необходимо Visual Studio 2019
Запустить проект ChatServer
Из браузера скопировать адрес страницы запущенной проектом ChatServer и заменить значение переменной urlServer в классе InteractionServer.
Сейчас значение  http://localhost:19028

Приложение получает, фильтрует и отправляет сообщения в формате json.
 
Принятые, отфильтрованные и отправленные сообщения отображаются ListBox 

Каждое сообщение имеет шаблон

"Пользователь"     "Дата отправки"
      "Текст сообщения"

В проекте реализованы три кнопки: Подключиться, Найти, Отправить.

1) При запуске проекта На сервер отправляется POST запрос, с телом json

{
  "filter": false,
  "dateStart": "2021-12-22T07:28:06.612Z",
  "dateEnd": "2021-12-22T07:28:06.612Z"
}

2) В ответ сервер отправляет Все сообщения которые были отправленны на сервер кнопкой "Отправить" (о ней дальше)
2.1) Все сообщения отображаются в ListBox 
2.2) Если серевер не доступен, в приложение выдаёт ошибку и кнопка Подключиться становиться активной.
2.3) По нажатию на кнопку Подключиться выполняется пункт 1.
3) Приложение отображает полученные сообщения в ListBox
4) Кнопка "Поиск" осуществляет поиск сообщенй по дате отправки. 
5) Кнопка поиск активируется установкой CheckBox "Фильтр" в положене true.
6) Выбирается  RadioButton "Равная" или "Диапазон"
7) Выбирается DatePicker дата поиска сообщений.
7.1) RadioButton "Равная" активирует один DatePicker и поиск происходит только по выбранной дате.
7.2) RadioButton "Диапазон" активирует два DatePicker и поиск происходит в диапазоне выбранных дат.
8) Кнопка "Отправить" отправляет текст сообщения на сервер.
